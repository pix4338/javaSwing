# 로또 프로그램
## 1. 주요 기능 
(1) 로또 당첨 번호를 회차에 맞도록 조회할 수 있고,사용자에게 값을 입력받아 당첨 번호와 비교하여 당첨 여부를 확인할 수 있도록 함.    
(2) 로또게임 규칙에 맞도록 난수를 랜덤으로 생성할 수 있는 기능을 제작함.
